# VerifyAI Support Website

This is the official support website for VerifyAI, hosted on GitHub Pages.

## Website Structure

```
website/
├── index.html                  # English homepage
├── index-zh.html              # Chinese homepage
├── support.html               # English support page
├── support-zh.html            # Chinese support page
├── user-guide.html            # English user guide
├── user-guide-zh.html         # Chinese user guide
├── privacy-policy-en.html     # English privacy policy
├── privacy-policy-zh.html     # Chinese privacy policy
└── assets/
    └── css/
        └── style.css          # Stylesheet
```

## URLs

Once GitHub Pages is enabled, the website will be available at:
- `https://yourusername.github.io/image-fact-check/website/`

Or if configured with a custom domain:
- `https://yourdomain.com`

## Updating Content

To update the website:
1. Edit the HTML files directly
2. Commit and push changes to the repository
3. GitHub Pages will automatically rebuild and deploy

## Local Development

To preview the website locally:
1. Open any HTML file in your web browser
2. Or use a local web server:
   ```bash
   cd website
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000` in your browser

## Support URL for Apple Connect

Use one of these URLs for your Apple Connect support page:
- English: `https://yourusername.github.io/image-fact-check/website/support.html`
- Chinese: `https://yourusername.github.io/image-fact-check/website/support-zh.html`

Replace `yourusername` with your actual GitHub username.
